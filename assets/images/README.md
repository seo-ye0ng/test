# Images

## 개요

이 폴더는 서비스에서 사용하는 이미지, 일러스트, 로고, 빈 상태 이미지 등을 관리합니다.

## 권장 파일 형식

| 형식 | 사용 상황 |
|---|---|
| PNG | 투명 배경 이미지 |
| JPG/JPEG | 사진 이미지 |
| SVG | 벡터 일러스트, 로고 |
| WebP | 웹 최적화 이미지 |

## 네이밍 규칙

```text
img_내용_위치.확장자
```

예시:

```text
img_empty_childmanagement.png
img_profile_mypage.png
img_onboarding_home.png
img_logo_header.svg
```

## 관리 규칙

- 이미지 용량이 너무 크면 압축 후 업로드합니다.
- 같은 이미지를 여러 이름으로 중복 저장하지 않습니다.
- 로고와 일반 이미지는 구분해서 관리합니다.
- 민감정보가 포함된 캡처 이미지는 업로드하지 않습니다.

## 개발 전달 메모

- 웹에서는 WebP 변환을 고려합니다.
- 모바일에서는 1x, 2x, 3x 해상도 대응이 필요한지 확인합니다.
- 이미지 대체 텍스트가 필요한 경우 문서에 함께 작성합니다.
